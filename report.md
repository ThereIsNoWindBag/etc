# 보고서 구성
## 1. 요약(초록):  전체 분석 절차와 결론
## 2. 분석 주제: 문제의식, 알아보고자 하는 것
## 3. 데이터 선정
- 데이터 선정 이유
- 데이터 소개(출처, 다운로드 URL, 특성, 구성)
## 4. 분석: 코드, 표, 그래프, 결과 해석
- 분석 목적 및 방법
- 변수 검토 및 전처리
- 분석 및 해석
## 5. 논의
- 한계점, 비판점
- 추후 분석 방향

# 평가 기준
- 절차 준수
- 논리적 완성도: 논제와 연계되는가? 분석 결과에 대한 해석이 적절한가?
- 참신성, 시의성
- 정교성:  비판 요소 인식, 추가 분석, 깊이 있는 분석
- 코드의 완성도
